# Phase 3 Execution Plan: 카드 UI 개선 및 필터 정보 표시

## 개요
카드 레이아웃을 변경하여 필터 정보(예산, 지역, 환경, 계절)를 Key-Value 형식으로 시각적으로 표시합니다. "자세히 보기" 버튼을 제거합니다.

---

## 구현 전략

### Step 1: 카드 구조 분석
**파일**: `components/sections/featured-cities-section.tsx`

현재 구조:
```
[이모지 배경]
[도시명]
[상세 정보: 월 생활비, 인터넷, 노마드 수]
[좋아요/싫어요 버튼]
```

목표 구조:
```
[이모지 배경]
[도시명]
[필터 정보 섹션: 예산, 지역, 환경, 계절]
[상세 정보: 월 생활비, 인터넷, 노마드 수]
[좋아요/싫어요 버튼]
```

---

### Step 2: 필터 정보 섹션 추가
**파일**: `components/sections/featured-cities-section.tsx` (수정)

필터 정보 표시 위치: 도시명 아래, 상세 정보 위

필터 정보 포맷:
```
🏷️ 예산: 100~200만원
📍 지역: 수도권
🌿 환경: 자연친화, 카페작업
🌸 최고계절: 봄
```

Key-Value 형식으로 각 필터를 라인으로 표시 (Text 스타일, Badge X)
