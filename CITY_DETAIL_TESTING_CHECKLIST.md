# 도시 상세페이지 최종 테스트 체크리스트

## ✅ 기능 테스트

### 라우팅 & 네비게이션
- [ ] 홈페이지 도시 카드 클릭 → `/cities/부산` 이동
- [ ] 각 도시 카드 클릭 테스트 (7개 도시 모두)
  - [ ] 부산 (/cities/부산)
  - [ ] 대구 (/cities/대구)
  - [ ] 대전 (/cities/대전)
  - [ ] 전주 (/cities/전주)
  - [ ] 강릉 (/cities/강릉)
  - [ ] 제주 (/cities/제주)
  - [ ] 성남 (/cities/성남)
- [ ] 상세페이지 뒤로가기 버튼 클릭 → 홈페이지로 돌아감
- [ ] 존재하지 않는 도시 URL (/cities/unknown) → 404 표시
- [ ] 관련 도시 카드 클릭 → 해당 도시 상세페이지로 이동

### 데이터 바인딩 검증
- [ ] 도시명 정확하게 표시
- [ ] 이모지 정확하게 표시
- [ ] 생활비 정확한 값 표시
- [ ] 인터넷 속도 정확한 값 표시
- [ ] 노마드 수 정확한 값 표시
- [ ] 예산 정보 정확 (4개 선택지 중 1개)
- [ ] 지역 정보 정확 (7개 선택지 중 1개)
- [ ] 환경 정보 정확 (다중선택)
- [ ] 계절 정보 정확 (4개 선택지 중 1개)
- [ ] 설명 텍스트 정확
- [ ] 주요 특징 4개 모두 표시

### 좋아요/싫어요 기능
- [ ] 좋아요 버튼 클릭 → 숫자 증가
- [ ] 싫어요 버튼 클릭 → 숫자 증가
- [ ] 새로고침 후 데이터 유지 (localStorage)
- [ ] 다른 도시에서 독립적으로 작동
- [ ] 모바일에서 터치 가능 (44px 이상)

### 추천 도시 기능
- [ ] 같은 지역의 도시 추천됨 (또는 같은 예산대)
- [ ] 현재 도시 제외되어 있음
- [ ] 최대 4개까지만 표시
- [ ] 각 추천 도시 링크 정상 작동

## 📱 반응형 테스트

### 모바일 (320-480px)
- [ ] 텍스트 크기 가독성 좋음 (14px 이상)
- [ ] 버튼 크기 터치 가능 (44x44px 이상)
- [ ] 이모지 크기 적절 (100px 이상)
- [ ] 가로 오버플로우 없음
- [ ] 그리드는 1열로 정렬
- [ ] 통계 카드 1-2열
- [ ] 특징 카드 1열
- [ ] 여백 적절 (좌우 16px)
- [ ] 헤더 고정되어 있음
- [ ] 스크롤 부드러움

### 태블릿 (768-1024px)
- [ ] 텍스트 크기 적절
- [ ] 그리드 2열로 정렬
- [ ] 통계 카드 3열 또는 2열
- [ ] 여백 균형 잡혀있음 (좌우 32px)
- [ ] 레이아웃 정렬 깔끔

### 데스크톱 (1024px+)
- [ ] 최대 너비 제약 적용 (max-w-7xl)
- [ ] 가운데 정렬되어 있음
- [ ] 그리드 3-4열 정렬
- [ ] 여백 충분 (좌우 48px)
- [ ] 호버 효과 작동
  - [ ] 카드 호버: 그림자 확대, scale 증가
  - [ ] 버튼 호버: 색상 변경
  - [ ] 링크 호버: 색상 변경
